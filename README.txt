Cryptocurrency payment gateway for WooCommerce and BucksBus. Accept coins: Bitcoin, Ethereum, TRX and more.

== Description ==

Cryptocurrency payment gateway for WooCommerce and BucksBus. Receive coins directly into your [BuckBus account](https://merchant.bucksbus.com/).

= Cryptocurrencies supported: =

- Bitcoin BTC
- Ethereum ETH
- Tron TRX
- Litecoin LTC
- Polygon POL
- Tether USDT (ERC20)
- Tether USDT (TRC20)
- Tether USDT (POLY)
- USDC (ERC20)
- USDC (POLY)

== Installation ==

1. Activate the plugin
2. Visit Admin > WooCommerce > Settings > Payment
3. Visit the gateway settings for each cryptocurrency.
4. Complete API Key, API Secret and Webhook secret fields to finish the setup. The instructions for each field are provided.
